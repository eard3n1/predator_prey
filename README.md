# predator_prey
An ecosystem simulation with a web-based visualization built with **Flask**.

## Overview
This project simulates a simple ecosystem where prey & predators interact on a grid. The simulation mimics realistic behaviors such as:

- **Prey**: Wandering, reproduction buff & population density bottleneck
- **Predators**: Hunting, reproduction nerf & starvation mechanics

## Setup & Running
### Prerequisites
- Python 3.7+
- Flask

### Installation
```bash
pip install flask
```

### Run the Simulation
```bash
python main.py
```

Open your browser at: <a href="http://127.0.0.1:5000">http://127.0.0.1:5000</a>
